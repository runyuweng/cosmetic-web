## Installation
- Install git and node.js
- Run the following code in terminal:
  ```
  $ git clone https://github.com/wengwengweng/cosmetic-web.git
  $ cd cosmetic-web
  $ npm i
  $ npm run dev
  ```
- Open your browser and enter ` http://localhost:3000/`
